Course Description

Robotics and artificial intelligence have become part of our daily lives, it is now been used in the offices, homes and security agencies. This course gives comprehensive guild on robotics and the application of artificial intelligence in automating devices for smart homes, smart offices, smart farms, smart hospitals and smart factories etc., it also covers Internet of Thing (IoT), and gives indebt understanding on how electronics works and their application areas. The students will learn how to apply the knowledge in solving world problems, become innovative by creating problem solving robots that mimics human and creating automated machines (Robots).

  • Writing Materials

Benefits of the course

  • Understand automation
  • Use Internet of Things in solving world problems
  • Understand Drone technology and how to build and fly them
  • Understand 3D Printing Technology
  • Compete with other robotics experts both local and international
  • Create automated devices capable of replacing human efforts
  • Become innovative

Web Development

Date: 13th Feb - 24th Mar, 2025

online 15-18 years STEAM
Get in touch Start a free trial

Course Description

This course provides an introduction to the fundamentals of web development using HTML and CSS. Students will learn the basics of HTML syntax, including creating web pages, organizing content using semantic markup, and adding links and images. They will learn CSS syntax, including styling web pages using color, typography, layout, and responsive design techniques. They will also learn fundamental concepts of JavaScript programming language, including data types, variables, operators, control structures, functions, arrays, and objects. Through a combination of lectures, demonstrations, and hands- on exercises, students will gain practical experience in designing and building web pages that meet web standards and best practices.

This course is for ..

  • The course is suitable for anyone interested in learning the fundamentals of web development using HTML and CSS.
  • Beneficial for individuals looking to enhance their career prospects by gaining basic web development skills.
  • Ideal for individuals who want to create their own websites.
  • Open to beginners with no prior experience in web development.
  • Suitable for students looking to complement their existing skills in graphic design.
  • Perfect for anyone who wants to bring creative ideas to life.

Eligibility Criteria

  • Basic Typing Skills
  • Basic understanding of how the internet works

Resources you need for this program

  • Students should come with (A Laptop/tablet, notebook and Pen)
  • Exercises require computer with internet access
  • Windows 7, Windows 8, Windows 10
  • 2 GB of RAM minimum (4GB recommended)

Benefits of the course

  • Website Creation Skills: Students will learn how to create websites from scratch using HTML and CSS. This includes structuring web pages with HTML tags and styling them with CSS properties.
  • Enhanced Career Opportunities: In today's digital age, having basic web development skills is highly valued by many employers. By learning HTML and CSS, students can enhance their career prospects and open up new job opportunities.
  • Increased Creativity: With HTML and CSS, students can unleash their creativity and design visually stunning websites that stand out from the crowd.
  • Improved Problem-Solving Skills: Web development involves a lot of problem-solving. By working on real-world projects and troubleshooting issues that arise, students can improve their analytical and critical thinking skills.
  • Understanding of Web Standards: HTML and CSS are essential tools for creating websites that meet industry standards and best practices. By learning these technologies, students will develop an understanding of the importance of web standards and how to adhere to them.
  • Ability to Collaborate: Web development is often a collaborative process, requiring individuals to work together to create a cohesive and functional website. By working on group projects, students can develop teamwork and collaboration skills.
  • Flexibility and Adaptability: As web development technologies evolve, it's important to stay current and adapt to new tools and techniques. By learning HTML and CSS, students can develop a mindset of flexibility and adaptability, which will serve them well in any field.
  • Basic JavaScript Coding: Write basic JavaScript code using the fundamental concepts learned in the course.
  • Interactive Web Pages: Create interactive web pages using JavaScript and DOM manipulation.
